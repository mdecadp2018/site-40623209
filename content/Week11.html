Week11

前言:嘗試利用 Heroku 的免費服務建立各組的網際輔助設計網站

◎課程內容

1.介紹區域輔助設計網站

  • 規劃使用 IPv6 協定上網
  • 安裝 Ubuntu 18.04 Server (各組可以在電腦教室中, 以 Virtualbox 及 IPv4 網路進行練習)
  • 設定純 IPv6 網路
  • 安裝 Nginx
  • 安裝 Flask 與 bs4
  • 安裝 uwsgi 與 Plugin
  • 設定 Nginx
  • 設定 uwsgi 啟動
  • 設定隨開機啟動輔助設計伺服程式

2.各組自行討論專題

◎課堂心得

我們這組(bg2)已經決定好製作的題目了(機械手臂)，然而機械手臂有百百種，這週上課就是在討論要製作什麼類型的手臂，每個人依據自己有興趣的方面去蒐集資料，到下週大家整合討論出最終的手臂，並開始製作。
